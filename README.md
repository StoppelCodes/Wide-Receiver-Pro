# NFLDataScience

A repo filled with different pieces of a nfl data science personal project hosted in azure cloud

Data pipeline:
Data scraped using R with the public nflscrapR package. (https://github.com/maksimhorowitz/nflscrapR). All credit to all contributors for this open-source library.

Machine Learning:
Machine learning done in Python for more control over models 

GUI: 
GUI created in Visual Studio and hosted on azure cloud server
