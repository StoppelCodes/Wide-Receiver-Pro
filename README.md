# WideReceiverPro

A repo filled with different pieces of a nfl data science personal project hosted in azure cloud

Data:
Data downloaded from the repository https://github.com/nflverse/nflverse-data/releases/tag/pbp. Credit to all involved in this project inlcuding tanho63 who was the original poster. 
Other data used scraped using R with the public nflscrapR package. (https://github.com/maksimhorowitz/nflscrapR). All credit to all contributors for this open-source library.

Machine Learning:
Machine learning done in Python for more control over models. Models constantly being tested and changed for best performance

GUI: 
GUI created in Flask

Hosting:
Hosted on Azure Cloud
